# 🛒 Dokan Hisab

**Grow and manage your store digitally**

Dokan Hisab is a modern full-stack web application designed to help shop owners manage their business digitally. It simplifies inventory tracking, sales management, and financial monitoring in one powerful and easy-to-use system.

Built with modern technologies to ensure **speed, scalability, and a smooth user experience.**

---

## 🚀 Features

- 📦 **Inventory Management**  
  Track products, stock levels, and updates easily.

- 💰 **Sales Tracking**  
  Monitor daily sales and business performance.

- 🧾 **Invoice Generation**  
  Automatically create and manage invoices.

- 📊 **Real-time Dashboard**  
  View business insights and analytics in one place.

- 👥 **Customer Management**  
  Store and manage customer information.

- 🔐 **Secure Authentication**  
  Safe login and user authentication system.

- 📱 **Responsive Design**  
  Works smoothly on desktop, tablet, and mobile devices.

- ⚡ **Fast Performance**  
  Optimized for speed and efficiency.
